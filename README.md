# 🧠 CNN for CIFAR Image Classification

A deep learning project that implements a **Convolutional Neural Network (CNN)** for image classification using the **CIFAR dataset**.  
This project demonstrates how CNN architectures automatically learn visual patterns and classify images into multiple object categories.

---

## 📌 Project Overview

Image classification is one of the most fundamental applications of Deep Learning and Computer Vision.  
In this project, a Convolutional Neural Network (CNN) is built and trained to classify images from the CIFAR dataset.

### Objectives
- Build a CNN architecture from scratch
- Train the model on labeled image data
- Evaluate model performance
- Visualize training accuracy and loss
- Understand feature extraction using CNNs

---

## 🗂️ Dataset

The project uses the **CIFAR dataset**, a standard benchmark dataset for image classification tasks.

### Dataset Features
- Image Size: **32 × 32 pixels**
- Channels: **RGB**
- Number of Classes: **10**
- Total Images: **60,000**

### Classes
- ✈️ Airplane  
- 🚗 Automobile  
- 🐦 Bird  
- 🐱 Cat  
- 🦌 Deer  
- 🐶 Dog  
- 🐸 Frog  
- 🐴 Horse  
- 🚢 Ship  
- 🚚 Truck

## 🏗️ CNN Architecture

The Convolutional Neural Network includes:

- Convolutional Layers (Feature Extraction)
- ReLU Activation Function
- Max Pooling Layers
- Dropout Layers (Overfitting Prevention)
- Fully Connected Dense Layers
- Softmax Output Layer

The CNN automatically learns hierarchical image features such as edges, textures, and shapes.
